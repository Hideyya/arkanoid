# Arkanoid (учебный проект)

Небольшая браузерная игра Арканоид на JavaScript и HTML Canvas.  
Проект учебный: цель - отработать работу с Canvas, анимацией, обработкой событий и простой игровой логикой.

## Возможности
- Управление платформой
- Мяч отскакивает от стен и платформы
- Кирпичи "разбиваются" при столкновении
- Подсчёт очков (Score)
- Таблица рекордов (сохраняется в браузере через localStorage)

> Примечание: рекорды хранятся локально на устройстве (в localStorage), поэтому на другом компьютере/в другом браузере список будет другой.

## Режимы игры
В проекте есть несколько режимов сложности. Сложность отличается скоростью мяча.

## Управление
Пробел - запуск игры
Стрелки ← / → — движение платформы
